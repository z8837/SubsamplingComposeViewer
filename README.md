# SubsamplingComposeViewer

`davemorrissey/subsampling-scale-image-view`를 로컬에 포함해
Compose UI로 감싼 안드로이드 이미지 뷰어 샘플입니다.

## 프로젝트 개요

- 전체 화면 이미지 뷰어
- 줌, 이동, 회전 지원
- `SubsamplingScaleImageView` 기반 대용량 이미지 표시
